NP Hub Static HTML Demo

Đây là ví dụ local cho API mới. Nhấn nút bên dưới để mở form, sau đó bấm Tạo yêu cầu khi để trống các trường bắt buộc để kiểm tra validate.

Vị trí nút nổi mặc định góc dưới phải (right/bottom = 20px). Tuỳ chỉnh trong np-hub.js bằng setAttribute("right"|"bottom"|"left"|"top", ...).
